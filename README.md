# Monad Academy - Base Mini App

Master trading with risk-free practice and competitive challenges on Base.

## Features

- **Paper Trading**: Practice trading strategies with virtual capital
- **VS Mode**: 1v1 trading competitions with customizable parameters
- **Group Battles**: Large-scale tournaments with prize pools
- **Performance Analytics**: Detailed insights into trading performance
- **Profile & Achievements**: Track progress and unlock achievements

## Tech Stack

- Next.js 15 with App Router
- React 19
- TypeScript
- Tailwind CSS
- OnchainKit for Base integration
- Farcaster MiniApp SDK

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Environment Variables

Create a `.env.local` file with:

```
NEXT_PUBLIC_ONCHAINKIT_API_KEY=your_api_key_here
```

## Deployment

This app is optimized for deployment on Vercel or any Next.js-compatible hosting platform.
